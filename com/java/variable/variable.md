### 标识符
	java中对各种变量、方法和类等命名时使用的字符序列称为标识符；
	凡是自己可以起名字的地方都叫标识符；
	
	
	命名规则：
		数字不可以开头；
		严格区分大小写，长度无限制；
		不可以使用关键字（int、class等）、保留字，但是可以包含（int abc_class = 100 ）；
		标识符不能包含空格；
		
	
	命名规范：
		1.包名：多单词组成时，所有的字母要小写：aaa.bbb.ccc
		2.类名、接口名：多单词组成时，所有单词首字母大写：XxxYyy（大驼峰）
		3.变量名、方法名：多单词组成时，第一个单词首字母小写，第二个单词开始每个单词首字母大写：xxxYyyZzz（小驼峰）
		4.常量名：所有字母大写，多单词时每个单词用下划线连接：XXX_YYY_ZZZ
	
	
### 变量
	变量概念
		内存中的一个区域，区域中的数据可以在同一个类型范围内不断变化；
		变量是程序中最基本的存储单元，包含：变量类型、变量名、变量值；
		变量必须先声明，再使用；
		变量都定义在其作用域内，在作用域内，变量是有效的，出了作用域就失效了；
		不能在同一作用域内定义同名的变量；
		
	变量的基本原理
		int a = 1; //定义一个变量，类型是int，值为1
		a = 2;  // 计算机会通过变量a找到a指向的内存空间，然后将内存空间中的值改成2，此时a的内存地址不变
		会在计算机的内存空间中开辟一个空间，保存这个1；
		这个1有对应的内存地址；
		a就等于这个内存地址，也就是a指向1的内存地址；
		
	
	变量使用两种方式
		（1）先进行变量的声明，然后进行变量赋值
			int a;
			a = 60;
			
		（2）变量的定义
			int a = 1;
